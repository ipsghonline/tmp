## Target Audience

- Network Engineers
- Infrastructure Technicians
- Site IT Coordinators

## Prerequisites

- Network admin credentials for Impact environment
- Physical access to networking equipment
- Understanding of DNS and DHCP configuration
- Familiarity with VPN and firewall configuration

---

## Quick Reference

**Estimated Duration:** 2-4 hours per site

**Critical Dependencies:**

- DNS propagation (allow 24-48 hours)
- Firewall rule updates
- VPN configuration changes

**Escalation:** Contact Infrastructure Team for network-specific issues
