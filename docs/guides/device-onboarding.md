## Overview

This worksheet guides you through deploying a new Windows device from unboxing to full operational status in the Impact environment.

## The 7 Deployment Phases

| Phase                      | Description                               | Duration |
| -------------------------- | ----------------------------------------- | -------- |
| **1. Pre-Deployment**      | Verify prerequisites and gather materials | ~10 min  |
| **2. Initial Setup**       | Power on and network connection           | ~10 min  |
| **3. Device Registration** | Autopilot profile assignment              | ~15 min  |
| **4. ESP Phase**           | Apps and policies installation            | ~30 min  |
| **5. Verification**        | Confirm deployment success                | ~10 min  |
| **6. Manual Config**       | Additional configurations if needed       | ~10 min  |
| **7. Handoff**             | User training and documentation           | ~10 min  |

## Prerequisites Checklist

- [ ] New device (factory sealed or reset)
- [ ] Device serial number registered in Intune
- [ ] User account created in Azure AD
- [ ] Licenses assigned (Microsoft 365, etc.)
- [ ] Network access available

## Expected Applications

After successful deployment, these apps should be installed:

- Microsoft 365 (Word, Excel, PowerPoint, Outlook)
- Microsoft Teams
- OneDrive for Business
- FloorSight
- Adobe Acrobat Reader
- Company Portal
